# HadoopStreamingExample

 Для запуска примера на hadoop кластере:
* Переместите папку wordcount_streaming в директорию $HOME
* Установите права на запуск для run.sh (chmod +x wordcount_streaming/run.sh)
* Положите файлы, которые необходимо обработать в папку wordcount_streaming/data/input
* Выполните sh wordcount_streaming/run.sh
